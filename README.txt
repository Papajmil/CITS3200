HOW TO RUN WEB APPLICATIONS LOCALLY WITH NODEMON

1. Install Node.js

2. Open Node.js Command Prompt (not Node.js)

3. Install nodemon globally
	npm install -g nodemon

4. Navigate to Web App directory
	E.G. cd C:\Users\selfa\Documents\University\UWA Sem 6\CITS3200\Exam Q Database\webapp

5. Run npm install to download dependencies
	npm install

6. Run Web App with nodemon
	nodemon

7. View Web App through browser
	URL: localhost:3000


NOTE:
  If you add any new modules to the web app, install them via the Node.js command prompt. This adds the module to the 
  dependecies for the project.
